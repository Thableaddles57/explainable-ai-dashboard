
# Explainable AI Dashboard

An interactive web dashboard for visualizing and interpreting machine learning model predictions using SHAP (SHapley Additive exPlanations) and LIME (Local Interpretable Model-agnostic Explanations).

## Features

- **Model Agnostic**: Works with any black-box machine learning model.
- **SHAP Integration**: Visualize feature importance and impact on predictions.
- **LIME Integration**: Understand local predictions for individual instances.
- **Interactive UI**: Built with Streamlit for easy exploration.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

To run the dashboard, navigate to the repository directory and execute:

```bash
streamlit run src/xai_dashboard.py
```

Then open your web browser to the provided local URL.
